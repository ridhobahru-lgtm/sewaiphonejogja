Cara Update

Edit satu file saja untuk menambah artikel baru setiap hari.

01

Buka file data artikel

Edit file artikel-data.js lalu tambahkan objek artikel baru di posisi paling atas.

02

Isi judul dan isi artikel

Lengkapi judul, tanggal, ringkasan, kategori, gambar, dan beberapa paragraf isi artikel.

03

Simpan lalu refresh

Setelah disimpan, buka ulang artikel.html dan artikel baru akan langsung muncul.

Tips cepat: Tambah artikel baru di urutan paling atas agar otomatis jadi artikel unggulan.
